# Scroll Story

An immersive web experience that transforms image sequences into interactive scroll-based animations. This project converts a 240-frame animation into a cinematic storytelling journey where each scroll reveals the next frame, creating a fluid visual narrative that responds directly to user interaction.

## Features

- **Full-screen image display** with responsive design
- **Smooth frame transitions** as you scroll through 240 images
- **Progress tracking** with visual progress bar and frame counter
- **Optimized performance** with image preloading
- **Keyboard navigation** support (arrow keys, spacebar)
- **Touch/swipe gestures** for mobile devices
